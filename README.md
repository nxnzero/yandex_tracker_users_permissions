# yandex_tracker_users_permissions

Записывает права пользователей к очереди в формате: 

| Очередь    | Права доступа | Пользователь  |
|------------|---------------|---------------|
| TESTQUEUE  | CREATE        | Иван Иванов   |
| TESTQUEUE  | WRITE         | Иван Иванов   |
| TESTQUEUE  | READ          | Иван Иванов   |
